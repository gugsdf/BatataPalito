Como funciona os commits?

O git commit funciona como um "snapshot" (foto) ou um ponto de restauração do seu projeto, registrando as alterações preparadas (staging area) no histórico do repositório. Ele cria uma versão única e segura, contendo metadados como autor, data e uma mensagem explicativa, garantindo rastreabilidade e colaboração

1. git add

2. git commit -m "Mensagem do commit"
